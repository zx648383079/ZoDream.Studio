# ZoDream.Studio

一个音视频合成制作软件


主要功能
-------
	这是一个通过调用`winmm.dll`实现演奏的程序
	主要实现的功能是乐器演奏。
	不完善的功能是曲谱自动演奏。

更新
------
	新增对 `#``b``·` 的识别

	实现更好的演奏
